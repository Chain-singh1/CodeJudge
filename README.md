# CodeJudge

A full-stack competitive programming platform built with React, Node.js, and MongoDB.

## Features
- Code execution in JavaScript, Python, Java, C++ via Docker
- AI-powered coding assistant (Google Gemini)
- Contest system with real-time leaderboard
- Activity heatmap and streak tracking
- Admin panel for problem and contest management

## Tech Stack
Frontend: React, Vite, Tailwind CSS, Monaco Editor
Backend: Node.js, Express, MongoDB, Docker
Deployment: Vercel (frontend), Render (backend), MongoDB Atlas
